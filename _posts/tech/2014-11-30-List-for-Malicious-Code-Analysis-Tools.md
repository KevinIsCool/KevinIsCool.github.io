* [FileAlyzer](http://www.safer-networking.org/products/filealyzer/)   
文件分析工具，FileAlyzer提供对文件的基础分析（包括显示文件的属性、以十六进制转存形式显示文件内容等），以及能够解释常用文件内容：如文件资源结构（文本、图像、HTML、媒体资源、PE）。
